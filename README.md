Simulação de Ataques de Força Bruta com Medusa e Kali Linux

Neste projeto, foi realizada uma simulação de ataques de força bruta utilizando a ferramenta Medusa no Kali Linux, testando três protocolos: FTP, HTTP e SMB. O objetivo foi demonstrar como invasores tentam adivinhar credenciais explorando senhas fracas, além de compreender medidas de defesa.

Ataques de força bruta são técnicas que usam tentativas repetidas e sistemáticas para descobrir senhas ou chaves. Apesar de antigos, ainda são muito eficazes quando encontram credenciais mal configuradas.

Existem várias variações desse tipo de ataque:

Força bruta simples: Tentativas manuais ou automáticas usando combinações óbvias (“123456”, “senha”).

Ataque de dicionário: Utiliza listas de palavras prováveis como tentativa de senha.

Ataque híbrido: Mistura dicionário com variações automáticas, como números adicionados ou substituição de caracteres.

Password Spraying: Testa poucas senhas comuns em várias contas para evitar bloqueios.

Força bruta reversa: Parte de uma senha conhecida (de vazamentos), tentando encontrar o nome de usuário correspondente.

O projeto mostra, na prática, como esses métodos funcionam na superfície e ressalta a importância de senhas fortes, autenticação multifator e políticas adequadas de bloqueio para mitigar ataques.

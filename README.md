# Projeto-Ataque-de-Brute-Force-de-Senhas-com-Medusa-e-Kali-Linux
Este laboratório foi criado para entender, testar e documentar métodos de autenticação insegura usando ambientes vulneráveis com o Kali Linux, a ferramenta Medusa e aplicações intencionalmente frágeis. O foco é aprendizado seguro e ética na cibersegurança, não a exploração real.
